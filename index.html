发表于 2021-08-07 分类于 足球

　　梅西离队的消息最终还是尘埃落定，事情没有像去年四月一样峰回路转，队史最佳以被球队背刺的方式离开效力二十一年的球队，巴萨向全世界展示了什么叫“好人就得让人拿枪指着”。罗马不是一天建成的，但要毁掉它，一夜大火就已足够。失去了克鲁伊夫的指引，拉波尔塔表现得就像离开塞内加的尼禄，无能而且冷血。

昆明：秋游春城

发表于 2020-09-15 分类于 旅行

　　这次去昆明，其实主要是为了工作。所幸会议的日程安排并不算太紧，我还有时间在昆明走马观花地逛一下。首先特别吐槽一下云南健康码不支持支付宝的问题。像我这样微信至今没有绑定手机号的人，疫情期间在昆明独自外出简直寸步难行。在此实名感谢怡帆多次替我填健康申报，让我有了自由活动的权利。

　　昆明机场落地后，首先面对的就是淅淅沥沥的小雨。好在这次并不需要在雨中去挤公共交通。把行李丢到宾馆之后，我们迅速在附近选定了当晚觅食的目的地——南亚风情街。

　　不知道是不是自己此前对“南亚风情”的理解有点偏差。初到南亚风情街的我开始有点恍惚。街边拥挤的小吃摊，挂着各种不知名品牌的廉价服饰店，散发着世纪初气息的招待所，甚至坑坑洼洼满是积水的路面，到处都是我这个成长在十八线城市的小镇青年所熟悉的元素。与一街之隔的高档商业区相比，时间在这里仿佛停滞了二十年。若是餐厅的物价也能如街景一般饱含年代感，这里其实也不失为一个理想的消费场所。

扬州：江都的膜法旅行

发表于 2017-10-13 分类于 旅行

　　从上海到扬州的旅途并不轻松。很难相信这座曾经江南最繁华的城市直到现在都没有一条直接与上海连通的客运铁路线。好在镇江站转城际大巴的换乘设计堪称便捷，通往扬州的旅程也谈不上艰辛。

　　扬州有过很多辉煌的称谓，从广陵到江都不一而足。在某位长者的自我奋斗和历史行程的影响下，如今这座江南名城俨然已经成为了一种信仰的发源地。作为虵的出生地，扬州就是众多蛤丝眼中的膜教麦加。在这里有必要说明一下，作为一名合格的共产党员，我一直都严格遵守党员一律不得信教的要求。江来报道上出现偏差，我可没法负泽。

　　最近江浙地区一直在下雨，气温也降到了十几度。为此我特意带了一条牛仔裤以备不时之需。然而就在我准备从旅店出发时，一个残酷的事实让我扬州之旅的第一个项目从瘦西湖游览变成了商业街购物——我居然忘了带腰带。。。穿着没有系腰带的牛仔裤，我在凄风冷雨的大街上提心吊胆地度过了在扬州的第一个小时。从强忍笑意的导购手中接过腰带，我用最快的速度匆匆系上，立马付钱走人。善良的我实在不忍心让她这么继续憋着。

泉州：迷航的光明之城

发表于 2017-09-14 分类于 旅行

　　泉州之行其实很早就在我的waiting list里了。虽然这座曾经世界上最繁华的港口早已风光不再，勾连东西方的海上丝绸之路也沉寂许久，但泉州深厚历史的积淀还是一直保留到了现在。

　　从某种角度来看，泉州和扬州都是被历史抛弃的城市。随着海运和漕运的废弛，两座城市都在一定程度上出现了难以逆转的的衰落。这点从中国铁路网对两座城市或多或少的不友善就可见一斑。作为一个拥有八百多万人口的城市，泉州居然只有一个新修建的高铁站，且远离市区。这让我感到不可思议。

　　第一次乘坐泉州的公交车的体验一言难尽。K1路司机师傅的任性，纵使是体验过武汉公交狂野飙车的人，也一时难以接受。一小时的车程里，我在拥挤的车厢里不断旋转，跳跃，闭着眼，身体无助地随着一次次的刹车和加速一起摇摆。虽然K1是空调车，但是当我下车时，身上的衣服早已湿透。我觉得自己根本就是在参加体操团体赛，还是一小时不带休息，只能玩单杠和吊环的那种。

　　泉州市区的街道密度要高于国内的大多数城市，但是道路宽度普遍偏窄，多数是双向双车道，晚高峰时候的拥堵现象极其严重。大概是因为开车太堵，泉州街上小电驴的数量秒杀国内绝大多数城市（不过后来我发现福州街上的电动车也数量惊人，所以这大概是福建特色？）。泉州市民骑小电驴的路数和公交司机的驾驶风格绝对是一脉相承，突出一个“浪”字。作为一名久经考验的（自行车）老司机，我头一回在街上感到紧张。其实我倒不是对自己没信心，但我真的信不过这些不按套路来的大哥和大姐们。。。

柏林：破碎的日耳曼尼亚之梦

发表于 2016-09-10 分类于 旅行

　　与西欧的一票千年古城相比，柏林只能算是一座年轻的城市。但历史的短暂并不代表经历的贫乏。作为一座见证了近代欧洲几乎所有重大变迁的城市，柏林自有独到之处。

　　喜欢独自旅行并不仅仅是因为自己享受自由的感觉。在柏林第一次乘坐轨道交通我就坐反了方向，如果有人同行，我肯定要乖乖背锅。但是一个人的话，这种无关紧要的失误就可以一带而过，这可是只有一个人独自行动才会拥有的特权。不过柏林地铁站台的风景还不错，坐错方向也不失为一次独特的体验。柏林地铁既没有闸机，也不设检票员。每个站台只配备几台售票机，乘车购票全凭自觉（当然了，随机查票这种事情肯定会有）。见微知著，和恨不得给地铁闸机装三道门的巴黎相比，柏林在建设资本主义和谐社会的道路上无疑已经遥遥领先。
